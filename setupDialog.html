Connect to REDCap to export data dictionaries and data spreadsheets in NDA format.



The filesystem database backend uses directories with CSV spreadsheets as an input source and does not require a database connection. For simplicity the assumption is that the data dictionaries can be derived trivially from the header column names of the data spreadsheets. Exporting such a data dictionary should be avoided as there are no variable descriptions. Export the data spreadsheet for upload to NDA.
